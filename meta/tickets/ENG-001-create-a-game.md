# ENG-001: Create a Game

## Goal
A host can visit the site and create a new game, receiving a short join code to share with other players.

## Key Flows
- Home page has a "Create Game" action
- Creating a game generates a short, shareable join code (e.g. 6 alphanumeric characters)
- Host is taken to the join screen where they enter their name and team (same as any other player)

## User Verification
- Visit the site → create a game → a join code is displayed on screen
- The code is short enough to read out loud and type on a phone
