# ENG-002: Join a Game

## Goal
A player can join an existing game by entering the join code, their name, and choosing a team — and see themselves appear in a basic lobby.

## Key Flows
- Player navigates to the site, enters join code + name + team (Team 1 or Team 2)
- On success, they land in a lobby screen showing their own name under their chosen team
- Invalid or unknown join code shows a clear error

## User Verification
- Enter a valid join code + name + team → see lobby with own name listed under the correct team
- Enter an invalid join code → see an error message
